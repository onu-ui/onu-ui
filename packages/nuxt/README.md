## Setup
1. Add `@onu-ui/nuxt` dependency to your project

```bash
pnpm i @onu-ui/nuxt -D
```

2. Add `@onu-ui/nuxt` to the `modules` section of `nuxt.config.ts`

```typescript
{
  export default defineNuxtConfig({
    // ...
    modules: [
      // ...
      '@onu-ui/nuxt'
    ],
    // ...
    css: [
      // ...
      'onu-ui/dist/style.css'
    ]
  })
}
```
